# NPB Historical Database for OOTP

[한국어](README_KR.md)

NPB (Nippon Professional Baseball) historical database for Out of the Park Baseball.

## Coverage

- **Years**: 1936-2025
- **Leagues**: Central League, Pacific League
- **Farm League**: Ni-Gun League (Eastern/Western) 2008-2025

## Files

### stats/
- `historical_database.odb` - Main NPB database
- `historical_lineups.odb` - Game lineups
- `historical_minor_database.odb` - Farm league (Ni-Gun) data
- `historical_transactions.odb` - Transaction records
- Various CSV files (Teams, Master, Rosters, etc.)

### Other folders
- `logos/` - Team logos
- `ballparks/` - Ballpark files
- `uniforms/` - Uniform files (ballcap, jerseys, pants, socks)
- `schedules/` - Schedule files
- `fg_files/` - Facegen files

## Installation

1. Download the files
2. Place them in your OOTP installation directory
