# NPB 히스토리컬 데이터베이스 (OOTP용)

[English](README.md)

OOTP Baseball용 가장 완벽한 NPB (일본프로야구) 히스토리컬 데이터베이스입니다.

## 주요 특징

| 항목 | 수량 |
|------|------|
| 선수 | 8,165명 |
| 타격 기록 | 54,266개 |
| 투구 기록 | 24,951개 |
| 구종 데이터 보유 투수 | 349명 |
| 팀-시즌 | 1,032개 |
| 수록 기간 | 1936-2024 |

### 포함된 내용

- **완전한 통계**: 1936년 이후 모든 NPB 선수의 타격, 투구, 수비 기록
- **고급 지표**: DRS (수비 기여도), 파크팩터, 시대별 조정
- **투수 구종**: 349명의 투수에 대한 정확한 구종 데이터
- **포스트시즌**: 일본시리즈, 클라이맥스 시리즈, 올스타전
- **명예의 전당**: 투표 기록 완비
- **상**: MVP, 베스트나인, 골든글러브, 사와무라상, 신인왕
- **2군 리그**: 니군(팜) 리그 데이터 (2008-2024)
- **시각 자료**: 100개 이상의 팀 로고, 유니폼, 구장 파일

## 설치 방법

### OOTP 25/24

1. [Releases](../../releases)에서 `NPB_Historical_v1.0.0.zip` 다운로드
2. zip 파일 압축 해제
3. OOTP 설치 폴더에 복사:
   - Windows: `C:\Program Files (x86)\Out of the Park Developments\OOTP Baseball 25\data\historical\`
   - Mac: `/Applications/OOTP Baseball 25/data/historical/`
4. OOTP 실행 → 새 게임 → 히스토리컬 → **NPB** 선택

> **중요**: 히스토리컬 게임 설정 시 리그 이름을 반드시 **"Japanese Probaseball League"**로 설정하세요.

## 파일 구조

```
dist/
├── stats/
│   ├── historical_database.odb      # 메인 선수 데이터베이스
│   ├── historical_lineups.odb       # 경기 라인업
│   ├── historical_minor_database.odb # 2군 데이터
│   ├── historical_transactions.odb  # 트랜잭션
│   ├── Master.csv                   # 선수 마스터 데이터
│   ├── Batting.csv                  # 타격 통계
│   ├── Pitching.csv                 # 투구 통계
│   ├── Fielding.csv                 # 수비 통계
│   └── ...                          # 기타 CSV 파일
├── logos/                           # 팀 로고 (100개+)
├── ballparks/                       # 구장 설정
├── uniforms/                        # 팀 유니폼
├── schedules/                       # 시즌 스케줄 (LSDL 형식)
└── fg_files/                        # 페이스젠 파일
```
